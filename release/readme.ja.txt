 これはなに？
----------------------------------------------------------------------
  Javaで書かれたLitecoin等のsCryptコインのマイニングツールです。
  stratum+tcpのマイニングプールに対応しています。
  JMiNya は pooler/JMinerのハッシングと一部のコードを使っています。

----------------------------------------------------------------------
 特徴
----------------------------------------------------------------------

  - GUI
    採掘パラメータを簡単に設定できるGUIがあります。
    いくつかのプリセットプールがあります。(MONA/NYAN/LTC/BEER)
    
  - 完全なPureJava実装
    プラットフォームに依存しません。たぶん

  - Stratum+tcp マイニングプールに対応
    一般的なstratum+tcpマイニングプールに接続できます。
  
  - 最高に低速なハッシング
    このプログラムは実験的な物であり、JavaでしかもCPUによる超低速
    なハッシングを行います。
    速度は1コア当たり1kH/s〜2kH/sです。
    (GPUをお持ちの方は素直にcgminerかcudaminerを使ってください。)

----------------------------------------------------------------------
 使い方
----------------------------------------------------------------------

  1. jminya.jarを起動します。windowsの場合はバッチファイルを起動してください。
  2. ウインドウが開いたら採掘パラメータを設定します。
    Stratum Server address
    +-MiningPool - プリセットのプール・または手動設定を選択します。
    +-ServerURL  - マイニングプールのstratum+tcpサーバアドレスを指定します。
    
    Worker Setting
    +-Name     - ワーカIDを指定します。
    +-Password - ワーカパスワードを設定します。

    Miner setting
    +-Number of Setting - 採掘スレッド数を設定します。
  3. "Start Mining"ボタンを押すと採掘がはじまります。
